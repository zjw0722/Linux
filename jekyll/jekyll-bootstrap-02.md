更换主题
---
主题放在`./_includes/theme/THEME-NAME`目录中，

当你安装了主题之后，你还可以通过`rake task`来更换使用你已经安装的主题
``` 'bash'
1. $rake theme:swith name="the-program"
2. #for 0.1.0 users 'rake switch_theme' still works
```
